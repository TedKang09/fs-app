🏢 OpenDart API 프로젝트

📋 프로젝트 개요

OpenDart API를 사용하여 회사 정보를 조회하고 다운로드하는 프로젝트입니다.

🚀 시작하기

다음 단계를 따라 프로젝트를 설정하세요:

  1. API 키 설정: config.env 파일에 OpenDart API 키를 입력
  2. 로컬 서버 시작: python -m http.server 8000
  3. 브라우저에서 접속: http://localhost:8000

📁 파일 목록

  • index.html - 메인 페이지 (현재 페이지)
  • config.env - API 키 설정 파일
  • api-test.html - API 테스트 페이지
  • company-downloader.html - 회사코드 다운로더

🔧 도구

💡 팁

API 키 발급: OpenDart 공식 사이트에서 API 키를 발급받으세요.

CORS 문제: 브라우저에서 직접 파일을 열면 API 호출이 차단됩니다. 반드시 로컬 서버를 사용하세요.
